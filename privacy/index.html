Privacy Policy

Last updated: February 15, 2026

Prompt Navigator for ChatGPT ("the extension") helps users navigate their own prompts in ChatGPT conversations. This policy explains what data the extension handles and how it is used.

Data the Extension Handles

The extension reads text content on supported ChatGPT pages (chatgpt.com and chat.openai.com) to identify and list user prompts for navigation.

The extension also stores UI preferences in chrome.storage.local, such as:

  • panel collapsed/expanded state
  • filter text
  • panel position

How Data Is Used

Data is used only to provide the extension's user-facing features: indexing prompts, filtering prompts, and scrolling to selected prompts in the current conversation.

Data Sharing and Selling

The extension does not sell personal data. The extension does not transfer chat content or stored UI preferences to the developer's servers or to third parties.

Data Storage and Security

UI preferences are stored locally in the browser via chrome.storage.local. The extension does not include remote code execution and does not fetch external scripts for runtime execution.

Data Retention

Local extension data remains on the user's device until removed by the user (for example, by clearing extension storage or uninstalling the extension).
